## ファイル構成

| ファイル | 内容 | 用途 |
|----------|------|------|
| `creative_brief.md` | 戦略ドキュメント — サービス概要・競合比較・ターゲット・訴求軸13個・ビジュアル方針 | 設計図作成時の参照元 |
| `regulations.md` | レギュレーション集 — Must/Wantルール・NG事例・ロゴ規定・実績データ・QC修正フィードバック | QC判定・コピーチェック・注釈確認 |
| `RAG CRレビュー ガイドライン (1).docx` | CRレビューガイドライン原本（Word） | regulations.md の原本 |
| `RAG_静止画制作ブリーフ｜リクルートエージェント【統合版】 (2).xlsx` | 制作ブリーフ原本（Excel） | creative_brief.md の原本 |

---

## 更新ルール

1. **regulations.md** — QCアプリで修正が発生するたびに「QCアプリからの修正フィードバック」セクションに追記。繰り返すパターンは本則に昇格
2. **creative_brief.md** — 訴求軸の追加・変更、ターゲット設定の更新時に改訂
3. **クライアントから新資料を受領** → このフォルダに配置し、README に行を追加

  creative_brief.md               戦略・訴求軸

       ↓ 設計図作成の参照元

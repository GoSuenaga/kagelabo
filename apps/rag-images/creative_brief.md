## 01. サービス概要と実績データ

| 項目 | 内容 |
|------|------|
| ブランド名 | リクルートエージェント |
| カテゴリ | 転職エージェントサービス（ハイキャリア層特化） |
| 実績 | 転職支援実績No.1（厚生労働省「人材サービス総合サイト」2023年度実績） |
| 求人規模 | 公開+非公開 90万件超。非公開単体 21万件以上（2025年12月時点） |
| 年収アップ実績 | 転職決定者の35.8%が年収1割以上増加（2024年10-12月期） |
| ミッション | 「一人ひとりの可能性に寄り添い、選択を支える」 |
| サービスの本質 | 専任アドバイザーによる伴走型サポート。書類添削・面接対策・年収交渉代行まで一括対応 |

## 02. 競合比較・差別化ポイント

| 競合 | 差別化 |
|------|--------|
| vs ビズリーチ | ビズリーチは「スカウト型」で自分から動く必要。リクルートエージェントは「伴走型」で受け身でもプロが動く |
| vs doda | dodaは間口が広いがアドバイザーの業種別専門性が低い。リクルートエージェントの業種別専門アドバイザー制度が強み |
| vs JAC | JACは外資・グローバル特化。リクルートエージェントは全方位対応で求人数圧倒的 |

**絶対的強み:**
1. 求人規模No.1（90万件超）
2. 年収交渉代行など具体的な価値提供
3. 業種別専門アドバイザー制度
4. 全国17拠点
5. 専用システム「Personal Desktop」で在職中も活動しやすい

## 03. ターゲット設定

## 04. ターゲットインサイト

| ID | インサイト | 詳細 |
|----|-----------|------|
| A | 市場価値の確認欲求 | 「自分は今の会社以外でも通用するのか？」確かめる機会がない。「転職を決意」ではなく「自分の価値を知りたい」が先 |
| B | FOMO（知らなかった損） | 非公開求人21万件の存在を知らない。「知っていたら選択肢が全然変わっていた」→選択肢の差を訴求 |
| C | 失敗への恐怖 | 「今より悪くなるかも」→「プロが一緒なら失敗しない」で安心・信頼を訴求 |
| D | エージェントへの警戒感 | 「しつこい」「利益第一」というイメージ→「あなたのペースで」「伴走型」でバリア解消 |
| E | 同世代比較欲求 | 同年代・同職種の年収が気になる→「自分は適正な報酬を得ているか？」で共感訴求 |
| F | プライドと承認欲求 | 年収600万以上層はプライドが高い。「あなたはハイキャリア層だ」と前提として扱い自己肯定感を刺激 |

## 05. クリエイティブの目的と感情設計

**最終KPI:** CV＝面接申し込み件数（年収600万円以上ユーザー）

**ゴール:** 「このサービスを使えば、自分のキャリアが動き出す」と直感させる1枚

| # | 感じてほしい感情 |
|---|-----------------|
| 1 | 「自分にも次のステージがあるかも」→ 前向きな期待感 |
| 2 | 「今より良くなれる。プロが一緒なら失敗しない」→ 安心と信頼 |
| 3 | 「知らなかったことで損をしていた」→ 気づきと行動欲求 |
| 4 | 「このサービスは自分のためのものだ」→ 自己肯定と共感 |

---

## 06. クリエイティブ方針

| 項目 | 内容 |
|------|------|
| コンセプト | 「プロと勝ち抜く転職」— プロと一緒に、ハイキャリアの次のステージへ |
| トンマナ | プレミアム・知的・信頼感・前向き。暗くなりすぎず、焦り・脅しを煽らない。ラグジュアリー感と親しみやすさを両立 |

**主要訴求軸:**
- A: 市場価値・キャリア証明
- B: 非公開求人（選択肢の差）
- C: 年収交渉力
- D: 業種専門アドバイザーの実力
- E: 同世代比較・競争心

**タブー＆NG:**
1. 画像内にテキスト・文字・ロゴを入れない
2. 「焦り・脅し・不安を煽る」表現はNG
3. 安っぽい・チープな印象の素材・配色はNG
4. ターゲットを軽く扱う表現はNG

## 07. ビジュアル方針（画像制作ルール）

### Type A: 人物系
1. 冒頭に「日本人のポートレート。」と明記
2. 役職/年収/性別を具体指定（例：ITコンサル・年収750万・男性）
3. 年代・前職・経験年数を描写
4. シネマティックなライティング
5. 「広告のキービジュアルのような写真」
6. 「文字情報一切無し」
7. 縦型9:16

### Type B: モノ・コンセプト系
1. 清潔感・フラット・ミニマルなトンマナ
2. 「真俯瞰のカメラアングル」を指定
3. 「中央から上部にかけてのスペースを広告コピーが入るよう大きく開けた構図」
4. 「文字情報一切無し」
5. 縦型9:16

**推奨カラーパレット:** ダークネイビー / ゴールド / オフホワイト / グレー系

**画像内テキスト:** 絶対に入れない。コピーは制作工程で後乗せ

## 訴求軸マスター（AX-01〜AX-13）

| ID | 訴求軸名 | ターゲット心理 | コピー方向性 | ビジュアル方向性 | 相性の良いセグメント |
|----|---------|--------------|------------|----------------|-------------------|
| AX-01 | 市場価値の可視化 | 自分の市場価値が今の会社の評価だけで正しいのかわからない | 「あなたの〇〇、本当に適正ですか？」疑問型で自己認識を揺さぶる | データ・グラフ・天秤など可視化系 | 全職種、特にコンサル・金融・エンジニア |
| AX-02 | 機会損失の恐れ | 動けるタイミングを逃したくない | 「〇〇の転職適齢期、意外と短い。」タイミングの緊急性 | 砂時計・チャンスの窓・上昇グラフ | マーケ・金融・30代全般 |
| AX-03 | プロへの信頼 | 一人で転職は情報格差があって不安 | 「〇〇の転職は、プロと動くのが正解。」断言型 | アドバイザーとの対面・握手・サポート象徴 | 製造業・エンジニア・初転職層 |
| AX-04 | 非公開求人の優位性 | もっといい案件があるのではと薄々感じている | 「あなたが見ている求人は、市場の一部に過ぎない。」事実提示 | 氷山・地図・隠れた宝 | コンサル・金融・全ハイキャリア |
| AX-05 | リファラルとの差別化 | 知人経由で本当に最適解なのか不安 | 「リファラルは、市場の氷山の一角に過ぎない。」客観的事実 | 氷山の一角・ネットワーク図 | コンサル・ハイキャリア |
| AX-06 | 年収交渉代行 | 年収交渉が苦手、言いづらい | 「言いにくい年収交渉、すべて丸投げでいい。」解放感 | 荷物を渡す・丸投げ・軽快ユーモア | 全職種 |
| AX-07 | 業種特化（コンサル） | 業界を知らないエージェントには任せられない | 「コンサルの転職は、コンサルを知るプロへ。」専門性断言 | チェス・数式など知的ビジュアル | コンサル専用 |
| AX-08 | 業種特化（金融） | 金融のキャリアパスは外部からわかりにくい | 「銀行・証券からの転職、狙い撃ちで最適解へ。」精度 | 金融街・高層ビル・ゴールドトーン | 金融専用 |
| AX-09 | 業種特化（製造業） | スキルが評価されるか不安、情報が少ない | 「製造業出身者の転職、一人で動くのはリスクがある。」リスク軽減 | 工場×オフィスのコントラスト | 製造業専用 |
| AX-10 | 業種特化（マーケ職） | 需要が高まっているが価値の活かし方がわからない | 「マーケターの転職、今が動くべきタイミング。」市場需要根拠 | 上昇グラフ・明るいオフィス | マーケ専用 |
| AX-11 | 実績・ブランド信頼 | 大手エージェントのメリットが知りたい | 「50年の実績が、あなたの転職に答えを出す。」数字で権威性 | ブランドカラー（赤×白）・タイポグラフィ | 全ターゲット |
| AX-12 | 第三者レバレッジ | 自己判断だけでいいのか、客観的視点が欲しい | 「キャリアの意思決定に、客観的な市場変数はあるか。」論理的 | 戦略図・論理的ビジュアル | コンサル・ハイキャリア |
| AX-13 | 社会的インパクト | もっと社会に貢献できる仕事がしたい | 「論理思考力を、もっと社会的インパクトのある現場へ。」志向 | 国家・都市・大規模プロジェクト | コンサル・社会貢献志向層 |

### 訴求軸の選び方ガイド（週次20本の内訳目安）
- AX-01〜02（市場価値・機会損失）：4〜6本
- AX-03〜06（プロ・非公開・リファラル・年収）：6〜8本
- AX-07〜10（業種特化）：4〜6本
- AX-11〜13（実績・第三者・社会貢献）：2〜4本
- 毎週ABテスト軸を1本必ず設定

### テスト優先順位
1. コピーのトーン（疑問型 vs 断言型）
2. ビジュアル（人物あり vs 概念ビジュアル）
3. ターゲットセグメント（職種別）
4. 訴求軸そのもの（全く異なるインサイト）
- 効果データを週次で必ずブリーフに反映

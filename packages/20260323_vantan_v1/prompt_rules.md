# 映像プロンプト作成ルール
# 2026-03-23 確定版

AI動画生成（Veo3）で映像プロンプトを書く際のルール。
実際のテスト結果から得られたフィードバックに基づく。

---

## 1. 実物がないものは指定しない

校舎外観など、実際の素材がないユニークなものを指定しない。
広告として正確性が大事。実物と異なるものは載せたくない。

代替: 汎用的な空間（明るいモダンな室内等）を使う。

## 2. 文字情報が映り込むプロンプトは避ける

看板、画面のテキスト、書類の文字、教科書など。
AIが変な文字を生成しやすく、文字が潰れて見える。

代替: 「教科書を開いている手元」→「ヘッドフォンをつけて白いノートにペンを走らせている肩越しショット」

## 3. 人物を出す場合は必ず「Japanese」「日本人」と明記

入れないと外国人が生成されやすい。

書き方: 「20歳の日本人女性」「a 20-year-old Japanese woman」を常に使う。
年齢はクライアント情報のデモグラから中央値を取る（16-24歳→20歳）。

## 4. 全カットに必ず動きの指示を入れる

食事俯瞰・風景など静止しがちなシーンでも動きを明示する。
完全に静止した映像はNG（絶対原則: Micro-movements）。

例:
- 「手がフォークに伸びる」→手指リスクあるので「湯気がゆっくり立ち上る」に変更
- 「カメラがゆっくりパンする」
- 「風で髪が少し揺れている」
- 「体重を少し移動させる」

## 5. ロゴ表示カットの動画プロンプトにロゴ指示を入れない

ロゴはCreatomateで後から重ねる。
動画プロンプトに「Logo」「ロゴ」と書くとAIが背景にロゴを生成してしまう。

代わりに「No text, no logos, no signage」を入れる。

## 6. 手指のクローズアップは避ける

AI動画生成では手指の描写が高確率で破綻する（指の本数が増える、関節がおかしい等）。

代替:
- 肩越しのミディアムショット（物は見えるが手は小さく映る程度）
- 後ろ姿のミディアムショット
- 足元のクローズアップ（足は比較的安全）
- 物だけのショット（人物なし）
- 湯気、光、風など自然現象で動きを出す

## 7. 映像の重複を避ける

11カットの中で同じような映像が繰り返されないようにする。
特に「スマホを操作する手元」は1動画に1回まで（CTAカットのみ）。

確認方法: 全カットの映像要素を絵文字で可視化して重複をチェック。

## 8. Veo3向けのEN定型句

全カットの末尾に必ず付与:
```
The camera has subtle micro-movements. The camera work is slightly shaky, as if shot on a handheld smartphone. The shot looks raw and unfiltered as if shot on an iPhone 13. Natural lighting, authentic social media aesthetic.
```

## 9. Veo3パラメータ

- duration: 4s（SNS広告で1カット4秒以上は不要）
- aspect_ratio: 9:16（縦型）
- resolution: 720p
- generate_audio: false
- モデル: fal-ai/veo3（標準版。fastより高品質）
- 1カットずつ順番に生成（並列だとタイムアウト）

## 10. Creatomate合成パラメータ

- テロップ位置: 画面中央（x:50%, y:50%）
- フォント: Noto Sans JP, Bold 900, 白, 影付き
- ロゴ: ○のカットのみ画面中央にオーバーレイ（テロップなし）
- トランジション: crossfade 0.1秒
- SE: 冒頭→カット1、商材名→スクール名カット、他→それ以外
- 出力: 720x1280, 30fps, mp4

# バンタン スプシ × Dify 動画ワークフロー やり方

## 仕組み

```
[Googleスプレッドシート] → [GAS] → [Dify API] → [動画URL] → [スプシに自動記入]
     ✓を入れてボタン押す     全データを渡す   Gemini/Veo3/ElevenLabs/Creatomate
```

- スプシの1行 = 1スクール = 1本の動画
- GASがスプシの全データを読み取り、Dify APIに直接渡す（Difyからスプシを取得しに行かない）
- ナレーション台本が空欄ならGeminiが自動生成する

---

## 1. Difyの準備

1. Dify で `vantan_fixed.yml` をインポート
   - 「スタジオ」→「DSLファイルをインポート」→ `vantan_fixed.yml` を選択
2. インポート後、右上「公開」→「更新」
3. 「APIアクセス」を開き、以下2つをメモ
   - **エンドポイントURL**（`https://xxx.dify.ai/v1` のような形式）
   - **APIキー**（なければ「作成」ボタンで発行）

---

## 2. スプレッドシートの準備

1. スプレッドシート `1yQHYqnhVG1rTiQKHjznMM4NBIP43e03p1dxJpy3rubc` を開く

2. **拡張機能** → **Apps Script** を開く

3. スクリプトを2つ作成（左の「+」→「スクリプト」）
   - `setup_vantan_spreadsheet` → 同名 `.gs` ファイルの中身をコピペ
   - `run_vantan_workflow` → 同名 `.gs` ファイルの中身をコピペ

4. `setupVantanSpreadsheet` を1回だけ実行
   - 初回は権限の許可を求められるので「許可」
   - 「運用データ」「API設定」の2シートができる

5. **API設定シート**に Dify の情報を入力
   - `DIFY_API_URL` の B列 → エンドポイントURL
   - `DIFY_API_KEY` の B列 → APIキー

---

## 3. 運用データの入力

「運用データ」シートに各スクールの情報を入力:

| 列 | 内容 | 必須 | 備考 |
|----|------|------|------|
| A: チェック | ✓ | - | 実行したい行に入れる |
| B: スクール名称 | バンタンゲームアカデミー | ○ | Difyの商材名になる |
| C: ターゲット | 20代女性 | ○ | プルダウン選択 |
| D: スクール概要 | ゲーム業界で活躍する... | △ | 台本自動生成に使う |
| E: 訴求ポイント | 業界直結のカリキュラム... | △ | 台本自動生成に使う |
| F: キーメッセージ | 夢を仕事に。 | △ | 台本自動生成に使う |
| G: 演出キーワード | 没入感/クリエイティブ | ○ | 動画の雰囲気を決める |
| H: 季節 | 春夏 or 秋冬 | ○ | プルダウン選択 |
| I: ナレーション属性 | 女性1 | ○ | プルダウン選択 |
| J: ナレーション台本 | （スラッシュ区切り） | - | **空欄ならGeminiが自動生成** |
| K: ロゴURL | https://... | ○ | jpg/png直リンク |
| L: 注釈 | ※個人の感想です | - | 画面下部に常時表示 |
| M: 反映率 | 50 | ○ | 0〜100（デフォルト50） |
| N: ステータス | （自動） | - | 実行中/完了/エラー |
| O: 動画URL | （自動） | - | 完了後に自動入力 |

△ = ナレーション台本が空欄の場合に必要（Gemini自動生成用）

---

## 4. 動画を回す

1. 「運用データ」シートを開く
2. 実行したい行の **A列に「✓」** を入れる（複数行OK）
3. メニュー **「Vantan」** → **「✓の行を実行」**
4. N列が「完了」、O列に動画URLが入ったら成功

---

## ナレーション台本の書き方

スラッシュ（`/`）でカットを区切る。10〜12カットが目安:

```
最近ちょっと気になってること/ゲームって実は仕事になるんだよね/バンタンゲームアカデミーって知ってる？/プロのクリエイターが直接教えてくれて/就職率もめっちゃ高いらしい/しかもチームでゲーム作れるの/毎日こんな感じで制作してて/めちゃくちゃ楽しそう/夢を仕事にするって最高じゃない？/気になったらチェックしてみて
```

空欄にすると D〜F 列の情報からGeminiが自動で台本を作ります。

---

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| 「DIFY_API_URL が未設定」 | API設定シートが空 | URLとKeyを入力 |
| Dify API 400エラー | 入力値の形式が合わない | B列（スクール名称）が空でないか確認 |
| Dify API 401エラー | APIキーが間違い | Difyで再発行して貼り直す |
| 動画URLが空 | Dify内部でエラー | Difyのログで確認 |

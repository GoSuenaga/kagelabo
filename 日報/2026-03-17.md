# 日報 2026-03-17

## やったこと

### QCギャラリーアプリ作成 (`qc-gallery-app/`)

広告クリエイティブのQCレビュー用Webアプリを新規作成した。

**構成:**
- バックエンド: Python FastAPI (port 8000)
- 画像生成: Google Gemini Imagen 3 API（当初fal.aiで作成→Geminiに変更）
- フロントエンド: SPA（gallery.html、ダークテーマ）

**作成ファイル:**
- `app.py` — FastAPIサーバー。`/api/regen?num=XX`でGemini API呼び出し→`./images/XX.jpg`保存
- `gallery.html` — カード型ギャラリーUI。Regen/OK/NG/Promptボタン、フィルター機能付き
- `briefs.json` — 20件の広告ブリーフ（prompt, headline, body, cta）
- `.env.example` — `GEMINI_API_KEY`の設定テンプレート
- `requirements.txt` — fastapi, uvicorn, google-genai, python-dotenv
- `start.sh` — ワンコマンド起動スクリプト
- `README.md` — セットアップ手順

**画像参照方式:** Base64埋め込みではなく`/images/XX.jpg`のパス参照

## 変更の経緯
- 最初はfal.ai (fast-sdxl) で実装
- 相談の結果、Gemini (Imagen 3) に変更。理由: 無料枠が大きい、APIキー取得が簡単、google-genai SDKがシンプル
